# zambian-head-rest
light-weight XR experience using Headrest from Zimbabwe / Zambian

Original asset: 1963.06.0013. Etnografiska museet, Världskulturmuseerna, Sweden. Headrest made of brown wood with ring and column as load-bearing parts on the foundation. Length 33 cm. Width approx. 8 cm. Rhodesia (Zimbabwe or Zambia).
